# SAD TO SIREN

Unity 기반 2D 로그라이크 생존 게임 프로젝트입니다.  
Unity와 C#을 독학하여 개발했으며, 아이템 제작, 인벤토리, 유닛 강화, 건물 시스템, 랜덤 맵 생성 등  
핵심 게임 시스템을 직접 구현하여 플레이 가능한 수준까지 완성했습니다.

## Overview

탐사 → 전투 → 자원 관리 → 성장으로 이어지는 반복 플레이 구조를 중심으로 설계된 게임입니다.  
랜덤 요소와 성장 시스템을 결합하여 로그라이크 특유의 플레이 경험을 구현했습니다.

## Core Features

- 아이템 제작 및 인벤토리 관리 시스템
- 플레이어 상태(체력, 스태미너) 및 시간 시스템
- 건물 설치 및 업그레이드
- 유닛 강화 및 장비 장착 시스템
- 랜덤 맵 생성 및 랜덤 루팅
- A* 알고리즘 기반 경로 탐색

## Tech Stack

- Unity
- C#

## Architecture

- 기능 단위로 Script를 분리하여 시스템별 책임을 구분
- Inventory, Unit, Building, Map 시스템이 상태를 공유하며 동작
- 플레이어 상태를 중심으로 각 시스템이 연동되는 구조

## Achievements

- 핵심 게임 시스템 전반 직접 구현
- 외부 에셋 없이 개발 진행
- 플레이 가능한 수준의 게임 완성 (early access 직전)

## Troubleshooting

### 육각형 타일 기반 A* 구현

기존 A* 알고리즘은 사각형 타일 기준으로 설계된 경우가 많아 그대로 적용하기 어려웠습니다.  
기존 알고리즘 구조를 분석하고, neighbor 탐색 방식과 거리 계산을 수정하여  
육각형 타일 환경에서도 동작하도록 구현했습니다.
