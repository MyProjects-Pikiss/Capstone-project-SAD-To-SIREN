# SAD TO SIREN

## Overview
Unity 기반 2D 로그라이크 생존 게임입니다.  
핵심 시스템을 직접 구현하여 early access 80% 수준까지 개발했습니다.

## Features
- 아이템 제작 및 인벤토리 시스템
- 플레이어 상태 및 시간 시스템
- 건물 설치 및 업그레이드
- 유닛 강화 및 장비 장착
- 육각형 타일 기반 A* 경로 탐색
- 랜덤 맵 생성 및 랜덤 루팅

## Tech Highlights
- A* 알고리즘을 육각형 타일 구조에 맞게 수정 및 적용
- 외부 에셋 없이 핵심 시스템 직접 구현
- Stable Diffusion 기반 리소스 생성

## Tech Stack
- Unity
- C#
